# Day-1
In this i learnt about how the internet works with the `HTTP` protocol.
In `HTTP` protocol we have 4 methods 
 - **GET** => In this method a client can retrieve data 
 - **POST** => It is a method where a client can submit something
 - **PUT** => It is a method where a client can update something
 - **DELETE** => It is a method to delete something from the db.
